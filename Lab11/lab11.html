本次Lab共有两部分：

  • Part A: 通过一个简单的例子来让大家熟悉HTML5的 Drag&Drop 和 File API
  • Part B: localStorage的使用，同时对Part A的程序做一些功能上的补充

Part A

本部分主要实现通过HTML5的Drag&Drop和File API实现一个简单的文件预览功能，效果示例参考截图。主要功能点包括：

  • 通过file input选择文件，然后点击preview按钮，下面会显示文件的基本信息以及预览内容
  • 通过将文件拖拽到预览区域，从而显示文件的基本信息和预览内容
  • 目前仅支持MIME type为image/*和text/*的文件预览，对于不支持的文件类型，预览框里显示文字提示不支持该格式的预览
  • 文件的基本信息包括：name，size，type，lastModifiedDate

Part B

随着Web的发展，在客户端存储一些数据很有必要。目前主要的解决方案有cookie，Web Storage，Indexed DB等，它们各有优缺点。在本部分，我们主要使用localStorage来实现文件预览的历史记录。效果示例参考截图，需要实现的功能为：

  • 每次预览一个文件的时候，表格中都会新增一项预览历史记录，主要记录文件的基本信息
  • 点击Clear History按钮的时候，清空所有的历史记录

注意： 由于localStorage中只能存储字符串，因此涉及到数组、对象等结构需要将其转成字符串。最好使用JSON.parse和JSON.stringify。
